# Pokemon Preview
This is a simple web app made with VanillaJS and static HTML + CSS.

The point of this was to get a bit of experience with programmatic UI in the browser.\
A secondary goal was to get literally anything REST API on my GitHub because I have only used them in closed-sourced projects.

It should be fairly responsive 

# How to use
Just type a name into the top search bar and press enter, it uses the pokeapi naming scheme so you can reference that if you don't know specific names.\
You can left-click a card to delete it and right-click to toggle a shiny form if one is available.
